FreeRTOS Task Mapping

UART Task
Receives start signal + credentials from MPU, sends PASS/FAIL, forwards app ↔ MPU data.
Wi-Fi Task
Configures Wi-Fi driver, monitors connection, manages lwIP netif & DHCP.
TLS Task
Performs mbedTLS handshake, manages TLS session, exposes read/write APIs to dispatcher.
Dispatcher Task
Routes data: UART ↔ TLS; handles retries, reconnection logic, watchdog pings.
Logger / Monitor
Non-blocking logging, status reporting, error counters.
